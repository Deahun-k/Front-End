! + tab 입력시 기본 시트 생성

<!---->입력 시, 자바스크립트 환경에서 주석처리 가능

id 속성이 clickBtn인 요소의 DOM 셀렉트
id 값이 같은 요소는 하나뿐이다.
(id는 하나 뿐이라 getElementById에서 s가 붙지 않지만, class는 s가 붙어서 getElementsByClassName로 사용한다.)

클릭이벤트가 발생하면 이벤트를 console에 출력.

색 지정시, "yello"와 같이 사용할 수 있지만, #ffff00; 로 사용할 수 있음.

문서의 거의 모든 요소가 객체가 될 수 있다.
함수를 변수에 참조 시킨다.
함수를 배열에도 담을 수 있다.
함수를 다른 함수의 인자로 사용.(callback 함수)

내부 선언 방식 (Internal)
외부 선언 방식 (External) - 외부 .js파일에 구현
<script src="./(파일이름).js"></script> 입력 시, 외부 js파일을 불러올 수 있음.

구현한 페이지에서 우클릭 -> 검사 또는, f12를 이용해 콘솔창 확인 가능.
